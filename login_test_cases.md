# Login Test Cases

1. Valid login  
Steps:  
- Enter valid email  
- Enter valid password  
- Click login  

Expected:  
User should login successfully


2. Invalid password  
Steps:  
- Enter valid email  
- Enter wrong password  
- Click login  

Expected:  
Error message should appear


3. Empty fields  
Steps:  
- Click login without entering any data  

Expected:  
Validation messages should appear
